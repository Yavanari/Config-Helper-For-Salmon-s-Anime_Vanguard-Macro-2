# Config Helper For Salmon's Anime_Vanguard-Macro-2
# Macro link
https://github.com/SalmonDXH/Anime_Vanguard-Macro-2
# How to download
 1. Go to [Config Helper download](https://github.com/Yavanari/Config-Helper-For-Salmon-s-Anime_Vanguard-Macro-2)
 2. Click green button **Code**
 3. Click **Download Zip**
 4. Extract all files to folder where your **(Salmon_AV.exe)  macro** located at
# Or just run UpdateHelper.ahk from pins
# How to use as a player
### Highly recomend go to "Popular configs" and find config you want here it will have right format to import
1. Press download config
2. Go back to main menu and select new folder as **Config folder**
3. Select **Gamemode , Map , Team** (Select Team only if original config had Team.txt in it)
4. Click Import Button
5. After importing **Config folder** will be deleted
### Other ways
1. Go to ‚Å†https://discord.com/channels/1309512479386243072/1388218189846151310 and download a custom config (only v2.0 configs work)
2. Copy the (.rar or .zip) file into the newly made ImportConfigs folder
3. Click **Extract All Configs** button(All Zip, Rar will be moved to ExtractedZIP folder)
4. Select **Extracted config folder** in config folder menu to search configs only in this folder(default means the ImportConfigs folder, use this only if the config you donwloaded contains only .txt files)
5. Select **Gamemode , Map , Team** (Select Team only if original config had Team.txt in it)
6. Click Import Button
7. After importing **Config folder** will be deleted
8. **Default Strategy** button will import universal default strategy from AIO V1 (made by :@knownaszad)

### Also you can manually select Strategy,Team,Coordinates txt in "Manual select config txts"
1. Browse all files in your explorer 
2. Select **Gamemode , Map , Team**
3. Click Import Button

# How to use as config maker
1. Select **Gamemode , Map , Team you want to export**
2. Click **Export** button
3. Will also ask if you want to export unit ability,mount,modifiers
4. Exported config will be saved in **ExportConfig** folder
